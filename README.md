This project analyzes trending YouTube videos in India to understand
engagement patterns such as likes, comments, video duration, and publish time.

- Matplotlib
- Seaborn
- YouTube Data API

Key insights:
- High views do not always mean high engagement
- Viral videos usually have better engagement ratios
- Shorter videos perform better
- Upload time affects trending visibility
